# sparkProject

SPARK product workspace.

## Apps

- `apps/sparkApp` - user-facing SPARK web app
- `apps/adminApp` - partner/admin app, planned

## Vercel

Deploy the user app with this root directory:

```txt
apps/sparkApp
```

When the admin app is added later, connect it as a separate Vercel project with:

```txt
apps/adminApp
```
